CSS

CSS is a used to add style to the html file like font, color,style.css means cascasding style sheets.
There are 3 diffrent ways the css can impliment.
  • internal CSS
  • external CSS
  • inline CSS

internal CSS

internal css is the css that is implimented inside the html file using sytle tag
for example:
        
            <style>
                body{
                    background-color:blue;
                }
            </style>
here it is an internal css that will give background color as blue.

external CSS

External css is the css that is implimented by adding and external .css file to the html file.

to use external css first you need to create css file with the css code and save as .css file and link to html using:
        
            <link rel="stylesheet" href="style.css" />

inline CSS

Inline css is the css that is implimented by adding css in the same line of the html code.
here is and example for inline CSS 
        
            <p style="background-color: aqua;"/>

Position Property

Position Property is the Property in CSS that is used to specify the type of Positioning
There are diffrent types of Position Property used.

they are:
  • Static
  • relative
  • fixed
  • sticky
  • absolute

static

in static position Property it does not affect the properties of up, bottom, left or right by default it is static. we can add static position Property by
        
            position: static;

relative

in relative position Property we can set values in top, right, left, bottom. it does not affect the other elements.
it can overlap the other elements 
        
            {
            position: relative;
            top:40px;
            left:30px;
            }

fixed

In fixed position Property the element is fixed on the page.we can assign the values like top,bottom,left,right.
if we scroll while we used fixed that element will stay at the postion. 
            
                position: fixed;
                top:50px;
                left:0px;

sticky

In sticky position property it the combination of static and fixed postion.it remains static until scrolled to specific postion. 
            
                position: sticky;
                top:0;

absolute

In absolute postion property it will go out of the flow.it is relative to the first positioned parent. 
            
                position: absolute;
                top:100px;
                right:40px;

Display property

display property is used to define the display of diffrent part in the httml page.
some of the dispalay properties are:
  • block
  • inline
  • inline-block
  • none

block

block is display property that the element will display as block element like p tag. it will display in new line. 
        
            display: block;

inline

the inline property in display property will display the elements in same line. the width and height will not have any effect.
lt only take the needed width 
            
                display: inline;

inline-block

the inline-block in the display property will display like the inline display property but in this we can adjust the height and width. 
            
                display: inline-block;

none

the none in the display property will hide the element. it will not show in the page. 
            
                display:none;

visibility property

visibility property is used to show or hide without changing the layout.
the types of visibility properties are:
  • hidden
  • visible
  • collapse

hidden

the hidden will hide the element in the element.but it will reconize and it will show space. 
                
                    visibility:hidden;

visible

visible is the default value. this will show the element.and there will be no change. 
                
                    visibility: visible;

collapse

the collapse in the visibility property will hide the row or column.but will leave the space in the table.
it is used in table tag. 
                
                    visibility:collapse;
