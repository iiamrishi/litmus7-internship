HTML TAGS


The HTML tags are the keywords that used to create webpage. the HTML tags usually have opening tag and closing tag and some does not have closing tags.
the syntax for HTML tags:

    <tag></tag>

!DOCTYPE

!DOCTYPE is an instruction that tells the web browser about the markup language in which the current page is written.It is used in HTML 5.

META TAG

meta tags are the tags that used to give information about the webpage. the meta tags are written in head tag.
these are some use of meta tag like
  • charset
  • viewport
  • description
  • author
an example of usage of meta tags :
    <meta charset ="UTF-8">

VOID ELEMENTS

void elements are the elements which only have opening tag and closing tag is not specified.
some of the void elements are :
  • <br>
    • the br tag is used to break the line.
  • <meta>
    • meta tags are the tags that used to give information about the webpage
  • <hr>
    • hr tag is used for horizonal line
  • <input>
    • input tag specifies an input field where the user can enter data.
  • <cols>
    • the cols tag specifies properties of column. it is used in table tag.
  • <img>
    • the img tag is used to insert image.

SEMATIC ELEMENTS

sematic elements are the elements which describes the meaning clearly. there are many sematic tags available in HTML.
some of them are:
  • <article>
    • the article tag is used to specify the article content
  • <footer>
    • this tag specifies the footer section
  • <header>
    • this tag is used for introductory section
  • <section>
    • the section tag defines the section in document
  • <nav>
    • the nav tag defines the set of navigation links

NON SEMATIC ELEMENTS

non sematic elements are the elements does not describe the meaning clearly unlike the sematic elements
example of non semantic tags are:
  • <div>
    • the div tag is used to make division in the content
  • <span>
    • the span tag is used to group elements for syling purpose.

LIST IN HTML

there are two types of list in html.
  1. orderd list
  2. unorderd list

ordered list is the list that is listed useing numericals like roman numericals or numbers.
