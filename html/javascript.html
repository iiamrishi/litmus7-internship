JavaScript Datatypes

There are some primitive datatypes in JavaScript.
they are:
  • string
  • number
  • bigint
  • boolean
  • undefined
  • symbols
  • null

String

strings are the datatypes which is characters.
example for string is :
let color = "blue";

number

the datatypes which are numericals are called number datatype.
for example:
let length = 10;

BigInt

bigInt is the datatype used for the large values
eg:
let x = BigInt(1234567899876543);

boolean

boolean is the type of datatypes which is used either true or false. it is mainly used for checcking conditions
for example:
let a = true;
let b = false;

undefined

undefined datatypes are when we dont give any values it is undefined.
for example:
let x;
since there is no value given to x it is now undefined.

symbols

symbols are used for creating uniqe identifiers. they are mainly used to create an object which is uniqe.
example:
const id = symbol('id');
here i created a constant id which is unique identifiers

null

null is nothing but giving without asign value
example:
let x = null;
